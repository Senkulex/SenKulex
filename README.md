# SenKulex

Лендинг независимой студии социальных и бизнес-проектов (Бельгия + Украина).
Сайт статический, размещён на **GitHub Pages**. Форма заявки отправляет данные в **Google Apps Script**, который пишет заявку, уведомляет владельца и отправляет автоответ пользователю **на его языке**.

🔗 **Live:** https://senkulex.github.io/SenKulex/

---

## Структура

| Файл | Назначение |
|------|------------|
| `index.html` | Весь сайт: разметка, стили, скрипты, переводы (UA / EN / NL / FR) — один файл |
| `Code.gs` | Серверная часть на Google Apps Script: приём формы, лог, письма |

---

## Как работает форма

1. Пользователь выбирает язык интерфейса (кнопки UA / EN / NL / FR).
2. При отправке формы на сервер уходит JSON:
   `{ name, email, message, language }`, где `language` — выбранный язык.
3. `Code.gs`:
   - проверяет поля,
   - (опц.) пишет заявку в Google-таблицу,
   - присылает **владельцу** письмо с указанием языка заявки,
   - присылает **пользователю** автоответ на его языке.

Поддерживаемые языки автоответа: `uk`, `en`, `nl`, `fr`. Неизвестный язык → fallback на `en`.

## Настройка серверной части (Google Apps Script)

1. Открыть [script.google.com](https://script.google.com) → **New project** → вставить код из `Code.gs`.
2. В начале файла заполнить:
   - `OWNER_EMAIL` — почта, на которую приходят заявки;
   - `SHEET_ID` — ID Google-таблицы для лога (или оставить `''`).
3. **Deploy → New deployment → Web app:**
   - *Execute as:* **Me**
   - *Who has access:* **Anyone**
4. Скопировать полученный `/exec` URL и вставить его в `index.html`
   в переменную `SENKULEX_SCRIPT`.

> ⚠️ После любой правки `Code.gs` нужно выкатывать новую версию:
> **Deploy → Manage deployments → ✏️ → New version → Deploy.**
> Без этого по старому URL работает прежний код.

## Деплой сайта

GitHub Pages: **Settings → Pages → Branch: `main` / root**.
Изменения в `index.html` публикуются автоматически после пуша в `main`.

## Лимиты

Бесплатный Gmail через `MailApp`: **до 100 писем в сутки**.
Для бóльших объёмов — Google Workspace (1500/сутки) или внешний почтовый сервис.
